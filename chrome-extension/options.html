COVE PM12 Daily Export

One-click sort, export, rename, and route Cove PM12 CSVs into your Drive-synced folder.

Configuration

Paste your full PM12 bookmark URL (with the assignee + status filter blob). The extension will replace any sortBy with Due Date ascending before opening it.
Path under your Chrome Downloads folder. Files land here as COVE PM12 YYYY-MM-DD.csv. Use slashes for nested folders, e.g. cove/csvs.

Setup checklist

  1. Make sure you're signed into manage.cove.is in this Chrome profile.
  2. Confirm the subfolder above is being mirrored to your Drive-synced COVE folder. Two ways to do this on Windows:
    • Symlink (recommended): open an Admin PowerShell and run
      New-Item -ItemType SymbolicLink -Path "$env:USERPROFILE\Downloads\cove-pm-csvs" -Target "<your COVE Drive path>"
    • Or set Drive Desktop to sync %USERPROFILE%\Downloads\cove-pm-csvs directly.
  3. Click the toolbar icon and press Download today's CSV. The file should land in your Drive folder within ~10 seconds.
